# JobPilot.AI ‚Äì Frontend

This is the **frontend application** of JobPilot.AI, built using **Next.js (TypeScript)** and **SASS**.  
It provides a **modern, responsive dashboard** to manage job applications and analyze resumes with AI.

---

## üöÄ Features

### Job Application Dashboard
- View, filter, and search job applications  
- Manage application statuses (Applied, Interview Scheduled, Offer Received)  
- Statistics and insights visualization  
- Full CRUD support for applications  

### Resume Management
- Upload and analyze resumes  
- Get AI-powered feedback and optimization suggestions  
- Track feedback history and maintain multiple resumes per application  

### Authentication
- Secure login/signup with protected routes  
- User session management with JWT tokens
